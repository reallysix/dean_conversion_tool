## 一、项目概述

**项目名称：** Dean Conversion Tool（语音转文字工具）
**技术栈：** SwiftUI + whisper.cpp + pyannote.audio
**平台：** macOS 14.0+ (Sonoma)
**构建系统：** XcodeGen + Xcode

### 核心功能
- 基于 Whisper AI 的本地离线语音转写
- 自动说话人识别（Speaker Diarization）
- 多格式导出（SRT / TXT / Markdown / HTML / JSON）
- 音视频文件支持（mp3, wav, m4a, mp4, mov 等）
- 视频文件内嵌播放器

### 3.4 遇到的问题及解决

#### 问题 1: NSApp 在 init() 中为 nil

**现象：** `Fatal error: Unexpectedly found nil while implicitly unwrapping an Optional value`

**原因：** `NSApp` 在 App 的 `init()` 阶段尚未初始化

**解决：** 将 `NSApp.appearance` 设置移到 `.onAppear` 中

#### 问题 2: 新文件未被编译

**现象：** `cannot find type 'SidePanel' in scope`

**原因：** XcodeGen 生成的项目文件需要重新生成

**解决：** 运行 `xcodegen generate` 重新生成项目

#### 问题 3: 导入按钮丢失

**现象：** 欢迎页面没有导入文件的入口

**原因：** `TranscriptToolbar` 只在有转写结果时显示

**解决：**
1. 将 `TranscriptToolbar` 移到 `TranscriptContainerView` 中始终显示
2. 在 `WelcomeView` 中添加醒目的「选择文件」按钮

### 3.5 最终调整：改为浅色方案

## 八、关键代码片段

### 8.1 Color(hex:) 扩展

```swift
extension Color {
    init(hex: UInt, alpha: Double = 1.0) {
        self.init(
            .sRGB,
            red: Double((hex >> 16) & 0xFF) / 255,
            green: Double((hex >> 8) & 0xFF) / 255,
            blue: Double(hex & 0xFF) / 255,
            opacity: alpha
        )
    }
}
```

### 8.2 说话人分组算法

```swift
var groupedBlocks: [(speaker: String?, segments: [TranscriptSegment])] {
    var blocks: [(speaker: String?, segments: [TranscriptSegment])] = []
    var currentSpeaker: String? = nil
    var currentSegments: [TranscriptSegment] = []
    
    for segment in segments {
        if segment.speaker != currentSpeaker {
            if !currentSegments.isEmpty {
                blocks.append((speaker: currentSpeaker, segments: currentSegments))
            }
            currentSpeaker = segment.speaker
            currentSegments = [segment]
        } else {
            currentSegments.append(segment)
        }
    }
    if !currentSegments.isEmpty {
        blocks.append((speaker: currentSpeaker, segments: currentSegments))
    }
    return blocks
}
```

### 8.3 波形占位符生成

```swift
private let heights: [CGFloat] = {
    let count = 80
    return (0..<count).map { i in
        let x = Double(i) / Double(count)
        let h1 = sin(x * 3.14 * 2) * 0.4
        let h2 = sin(x * 3.14 * 5 + 1.2) * 0.3
        let h3 = sin(x * 3.14 * 8 + 2.5) * 0.2
        let noise = sin(x * 3.14 * 13) * 0.1
        let value = abs(h1 + h2 + h3 + noise)
        return max(6, value * 36)
    }
}()
```
